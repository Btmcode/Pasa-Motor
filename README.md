# Paşa Motor

İstanbul Fatih'te TVS, Hero, Falcon ve Işıldar yetkili motosiklet bayii. Motosiklet satışı, servis, bakım ve yedek parça hizmetleri.

## Teknolojiler

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui
- React Router DOM
- TanStack Query

## Kurulum

```bash
npm install
npm run dev
```
